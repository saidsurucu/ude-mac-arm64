# ⌨️ Klavye Kısayolları (macOS)

UYAP Doküman Editörü aslında Windows için tasarlandığından, kısayolları `Ctrl`
tuşuna ve alışılmadık tuş birleşimlerine dayanır (örneğin kalın = `Ctrl+K`). Bu
Apple Silicon derlemesinde kısayollar, Mac'te beklediğiniz gibi **`Command (⌘)`
tuşuyla** ve mümkün olduğunca **Microsoft Word'ün Mac kısayollarıyla aynı**
çalışacak biçimde uyarlanmıştır. Düğmelerin üzerine geldiğinizde çıkan ipuçları da
(tooltip) artık Windows değil **Mac kısayolunu** gösterir.

> Simgeler: `⌘` Command · `⇧` Shift · `⌥` Option (Alt) · `⌫` Delete · `⌦` ileri-sil (`fn`+`⌫`) · `←↑↓→` ok tuşları

---

## Biçimlendirme

| İşlev | Kısayol |
|---|---|
| Kalın | `⌘B` |
| İtalik | `⌘I` |
| Altı çizili | `⌘U` |
| Tümünü BÜYÜK harf yap | `⌘⇧A` |
| Harf düzenini değiştir (küçük → BÜYÜK → Baş Harf) | `⇧F3` |
| Yazı özellikleri (font penceresi) | `⌘T` |
| Stil özellikleri | `⌘⇧R` |
| Biçim kopyala | `⌘⇧B` |

> Ayrı bir "küçük harf" kısayolu yoktur; küçük harfe `⇧F3` döngüsüyle ulaşırsınız
> (Mac Word'de de böyledir).

## Yazı boyutu

| İşlev | Kısayol |
|---|---|
| Yazıyı büyüt | `⌘⇧.` (Command+Shift+nokta) |
| Yazıyı küçült | `⌘⇧,` (Command+Shift+virgül) |

| İşlev | Kısayol |
|---|---|

| Türünü değiştir | `⌘⇧J` |
| İmzalar | `⌘⇧G` |

## Metin içinde gezinme

macOS'un her metin kutusunda beklediğiniz gezinme tuşları (Windows derlemesinde
çalışmaz, bu derlemede geri kazandırıldı):

| İşlev | Kısayol |
|---|---|
| Kelime kelime atla | `⌥←` / `⌥→` |
| Satır başına / sonuna git | `⌘←` / `⌘→` |
| Belge başına / sonuna git | `⌘↑` / `⌘↓` |
| Önceki kelimeyi sil | `⌥⌫` |
| Sonraki kelimeyi sil | `⌥⌦` (`fn`+`⌫`) |
| İmleçten satır başına kadar sil | `⌘⌫` |

> Bu gezinme tuşlarına `⇧` (Shift) eklerseniz **seçerek** ilerlersiniz
> (örn. `⌥⇧→` = sonraki kelimeyi seçer, `⌘⇧↑` = belge başına kadar seçer).

## Yakınlaştırma

| İşlev | Kısayol |
|---|---|
| Yakınlaştır | `⌘+` |
| Uzaklaştır | `⌘−` |
| Trackpad ile yakınlaştır/uzaklaştır | `⌘` + iki parmakla yukarı/aşağı |

---

### Notlar

- Aşağıdaki işlevlerin Mac kısayolu **eklenmemiştir**, çünkü bunlar araç çubuğunda
  düğme olarak değil, pencere/açılır-menü içinde bulunur ve güvenli biçimde
  klavyeye bağlanamaz: **Değiştir, TL Simgesi, Asılı Girinti, Numaralandırma değeri
  ayarla, Son Belgeler.** Bunları araç çubuğundan/menüden kullanabilirsiniz.
- `⇧F3`, `F7` (imzala), `F8` (mobil imza) gibi fonksiyon tuşları Mac klavyesinde
  gerekiyorsa `fn` ile birlikte basılır (ör. `fn`+`⇧`+`F3`) ya da Sistem
  Ayarları'ndan "F1, F2 vb. tuşları standart fonksiyon tuşları olarak kullan"
  seçeneğini açabilirsiniz.
- `Shift+Alt+…` ile çalışan ekleme komutları (Sembol, Tablo, Resim, Sayfa No,
  Alt/Üst Yazı) Mac'te de aynı şekilde **`⇧`+`⌥`+…** ile çalışır.
